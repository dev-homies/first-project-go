# dev-homies / first-project

Requirements:

- Docker

## Start Project

```
./scripts/up.sh
```

## Stop Project

```
./scripts/down.sh
```

## Reset Project

```
./scripts/destroy.sh
```

## View All Logs

```
./scripts/logs.sh
```

## View Service Logs

```
./scripts/logs.sh "db"
./scripts/logs.sh "api"
./scripts/logs.sh "web"
./scripts/logs.sh "pgadmin"
```
